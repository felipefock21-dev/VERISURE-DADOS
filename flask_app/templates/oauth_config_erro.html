OAuth não configurado (Erro 401 invalid_client)

O app ainda está usando credenciais de exemplo. O Google rejeita isso e mostra "The OAuth client was not found".

Faça o seguinte:

  • 1. Em Google Cloud Console → Credentials, crie um OAuth client ID do tipo Web application.
  • 2. Em "Authorized redirect URIs" adicione exatamente:
    {{ redirect_uri }}
  • 3. Copie o Client ID e o Client secret.
  • 4. Na pasta do projeto (onde está run_app.py), crie ou edite o arquivo .env com:
    OAUTH_CLIENT_ID=seu_client_id.apps.googleusercontent.com
    OAUTH_CLIENT_SECRET=GOCSPX-xxx
    DEPLOY_URL={{ deploy_url }}
  • 5. Reinicie o app e tente "Conectar ao Google Drive" de novo.

Veja também .env.example no projeto.
