Erro 400: redirect_uri_mismatch

O Google só aceita login se a URI de redirecionamento estiver exatamente cadastrada no seu OAuth Client.

URI que este app está usando (copie e use no passo 2):

{{ redirect_uri }}

Passos para corrigir:

  1. Abra Google Cloud Console → Credentials.
  2. Clique no seu OAuth 2.0 Client ID (tipo Web application).
  3. Em Authorized redirect URIs, clique em + ADD URI.
  4. Cole exatamente a URI que está em destaque acima (sem espaço no início/fim, com https:// e caminho /oauth2callback).
  5. Clique em SAVE.
  6. Volte ao app e tente fazer login de novo.

No Render (variável de ambiente):

Confirme que DEPLOY_URL está igual à URL do seu site, sem barra no final, por exemplo:

{{ deploy_url }}
